########################
License Management |Pro|
########################

*Fast DDS Monitor Pro* is distributed under the **eProsima Software License Agreement** and requires a
valid license to run. The application checks for a license automatically at startup.

.. _pro_license_check:

License Check at Startup
=========================

Every time *Fast DDS Monitor Pro* starts, it performs a license validation before opening the main window.
If a valid license is found, the application launches normally. If no license is detected, an error message
is shown and the application exits.
